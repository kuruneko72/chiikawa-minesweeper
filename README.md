# 🐰 乌萨奇扫雷 - Chiikawa Minesweeper

一个可爱的 Chiikawa 主题扫雷游戏！地雷变成了可爱的乌萨奇，整体风格是温暖的奶黄色。

## ✨ 特色功能

- 🐰 **Chiikawa 主题**：地雷替换为乌萨奇图标
- 🎨 **奶黄色友谊风格**：温暖的配色方案
- 🔊 **音效系统**：踩到乌萨奇时的爆炸音效、点击音效、胜利音效
- 🎮 **经典玩法**：左键揭盖，右键标记
- 📊 **三个难度**：初级 (9×9)、中级 (16×16)、高级 (16×30)
- ⏱️ **计时器**：记录你的游戏时间
- 📱 **响应式设计**：支持桌面和移动端

## 🎮 如何游戏

1. **左键点击**：揭开格子
2. **右键点击**：标记乌萨奇的位置
3. **目标**：揭开所有安全的格子，保护乌萨奇！
4. **数字**：表示周围 8 个格子中有多少只乌萨奇

## 🚀 本地运行

直接用浏览器打开 `index.html` 即可，或使用本地服务器：

```bash
python3 -m http.server 8080
```

然后访问 `http://localhost:8080`
